# Stamp

온라인 모각코 진행 후 인증 확인 페이지입니다.\
인증 기준은 아래와 같습니다.

- 10시 30분 ~ 10시 40분 사이에 자기 소개 혹은 진행 내용 소개하신 분
- 12시 30분 ~ 12시 40분 사이에 마이크 켜고 자유로운 대화에 참여하신 분
- 12시 40분 이후 부터 다음 모각코가 시작되기 전 까지 진행했던 내용을 issue comment에 업데이트 되고 난 이후 stamp 업데이트 진행
