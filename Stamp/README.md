# Stamp

온라인 모각코 진행 후 인증 확인 페이지입니다.\
인증 기준은 아래와 같습니다.

- 10시 ~ 10시 10분 사이에 자신임을 증명할 수 있는 스크린샷 찍기
- google meet 입장 후 카메라를 통해 찍은 스크린샷이어야 함
- 12시 이후 한 일에 대한 내용이 이번 모각코 issue 글에 추가되고 난 이후 이 페이지에서 인증 stamp 확인
