# Stamp

온라인 모각코 진행 후 인증 확인 페이지입니다.\
인증 기준은 아래와 같습니다.

- 10시 ~ 10시 10분 사이에 자기 소개 혹은 진행 내용 소개하신 분
- 12시 ~ 12시 10분 사이에 마이크 켜고 자유로운 대화에 참여하신 분
- 12시 10분 이후 진행했던 내용이 이번 모각코 issue 글에 추가되고 난 이후 이 페이지에서 인증 stamp 확인
